Website Development

This is a list of all planned and potential web development, more or less in the order it will happen. If you have more ideas or would like to see something done a little sooner than planned then send your ideas to the webmaster.

 

Item What and Why Requirements
Pic/Memory of the week/day on homepage A constantly changing image or story taken randomly from the Memory Lane section, designed to give a reason for people to return to the web page every so often (which in turn is to ensure they keep thinking about and looking forward to the celebration). Get a critical mass of material in Memory Lane so that the content will vary regularly (say 20 images and 10 stories)
Google map alumni mashup Google maps can be combined with address information to provide a world map where each alumni member has a marker representing their current location (to whatever resolution they are willing to share). Shows people how the influence of the band has spread around the world. Include parade route information for SOMF, and music centre location. Some background work required to figure out how to do this, then to figure out how to let people gauge how much detail they will share.
Google map performance mashup Similar to the above but adding a pin for every place the band has performed over the years. Secondary idea to allow members to type in any special memories local to that performance. Getting initial population data from Rob Bennett, or the collective memory of the committee.
Live Forum Set up interactive bulletin-board style forums where people can communicate with each other through the website. Need to find a suitable drop-in component and figure out how to hook it up to existing pages. Login required first as forums should disallow anonymous posting.
Fun and Games The web is chock full of little games and such you can post onto your own website. Alternatively we can come up with our own that are more BTTB-specific. The idea is that people not only come back to visit the website regularly but also enjoy doing it. Ideas could include crosswords, word-search, sudoku, trivia, picture identification, just-for-fun polls, band acronyms/sayings, etc. Need a variety of things ready to go before introducing it - possibly a commitment to populate any home-grown games
Committee link to send email to groups A web link for committee members which will open up a mail program with the email addresses of sub-groups of alumni (e.g. all those wanting to play in the concert band) pre-populated. Makes it easier to target any communications going out to those to which it pertains None - Can do any time
Music and Video Clips We have a collection of music and video clips that could be made available. Space restrictions on the website make it impractical to load everything at once but it should be possible to have a select "clip of the day/week" Finish the rearrangement of navigation buttons to allow for a music player on the front page.
Online purchase of event packages The ability to peruse the list of event packages we're going to offer for sale (e.g. the all-inclusive "Strike up the Band package" for $8 billion, which includes your very own band complete with uniforms, coaches, music director, and a lifetime pass to McDonalds). By having advance purchase available we'll be more comfortable with the budget we're setting. Need Boosters to set up PayPal - it's the best option since you can pay directly with a credit card now even without a PayPal account. Since we're a subcommittee they have to be the ones to set it up. Rates range from 0.9% to 2.9% of the transaction fee.

    Painting Orders Allow people to order paintings in advance, charging to PayPal or providing an order form for them to print and mail in with a cheque. Committee page should allow assignment of a print ID to orders, with allowances for reservation of certain low-number IDs for raffles, donations, etc. Need Boosters to set up PayPal.
    News database Keep track of front page news items in a database so that it can be updated by anybody, and so that we can hide all of the articles a returning alumni has already seen. Requires a login so that we can associate what they've seen with who they are.
    Pre-ordering of commissioned art Using the name alone and the subject of the prints there may be early interest in pre-ordering the work that John Newby will eventually get done for us. Since being able to sell all 450 prints is a high-risk proposition it's in our best interests to make it as easy as possible to get the art ordered, possibly offering some kind of discount depending on how far in advance you order (biggest before you actually see the work of course). Should roll out when event package purchase options are available (and be one of them). Can't be visible until price and final detail on the prints are available.
    Music Download Provide a method for people signed up for the parade and/or concert to download appropriate music parts. Susie will gauge the parade part coverage, make sure all parts are filled. People will a) request parts, giving their name, b) get approved by Susie, and c) be sent a link where they can download the sheet music. A similar task for the concert coverage will be given out to someone later. Must have all music converted from scans to PDF files.

Proposed changes take us beyond that so at some point it will be desirable to rearrange the navigation to make it simpler to find things. Simultaneously an image pre-load will be added to speed up the page in general. Items to be handled include:
  • Shrinking of the logo to provide more screenspace (with dynamic grow)
  • New graphics for the menu
  • Submenu capabilities so that everything can still be accessed from a single location
  • Login request at the bottom of the menu

    Alumni info editing Allow the alumni to edit their own information, including profile data, adding their own memories, allowing certain profile information to be visible to others.     Database Conversion Switch profile data from its current setup to be inside a live database. Will fix slow loading of home page, and allow the Memory Lane section to be created faster.     Login Allow people to enter the site with identification so that they can post things and we'll know who they are, or we know exactly what they have permission to change so that e.g. profile changes don't require our intervention.
