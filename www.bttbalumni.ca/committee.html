Committee Functions

Links for BTTB Alumni Committee use only.

link:(#dbQuery,Information queries) Provides a set of pre-coded queries on the database as well as a text field for random queries.
link:(#registration,Registration Spreadsheet) Provides an area to upload and download the registration spreadsheet.
link:(#paid,Registration Payment List) Check list to update the list of alumni who have paid for registration to the celebration events.
link:(#printChart,View print charts) Show the current list of John Newby prints sold, including those with specific numbers.
link:(#profiles?committee=1,Approve new members) This link will show you a list of current registrations and a list of those awaiting approval. Check the boxes of all that are valid and hit the "Approve" button to add them to the site.
link:(#bios, See Committee Bios) These are a work in progress that I won't make live until the committee members are okay with the image and information that appears with their name. Send me mail with changes that you wish to make.
download:(__CGIHREF__/getInfo.cgi,Download Alumni data) This will download a file in .txt format that contains all of the alumni data in tab-separated format suitable for importing into Excel or Word. This is potentially sensitive data so be careful what you do with it.
link:(#webwork, See Website development plan) This link will show you the list of planned and potential improvements to the website.
Anything else? Let send:(bttb@picott.ca,Kevin Picott) know what other functions would be useful to you and they can be added
